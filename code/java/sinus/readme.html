Sinus v1.0 -- Java Sinus Scroller Applet.

Copyright (c) 1996 Anthony Miceli.

The sinus applet included in this archive is a simple, visually attractive mechanism for smoothly scrolling text across the display. The text wraps around a sine wave as it scrolls to the left and changes colour as it oscillates vertically. To pause the scrolling at any time, click the left mouse button anywhere within the applet. Repeating this action resumes the scrolling. This applet will run best when viewed with a browser that implements a Just-In-Time compiler (such as Internet Explorer 3.0/Netscape Navigator 3.0). It was written using Symantec Cafe 1.50.

USAGE

To embed the applet into your HTML document, use the following syntax:

<APPLET CODE="sinus.class" WIDTH=www HEIGHT=hhh>
<PARAMETERS...>
</APPLET>

The following parameters are supported by this applet. All are optional:

NAMETYPEDESCRIPTION
messagetext stringThis is your message to be displayed. Note that a short message will follow your text.
yposintegerVertical position for the center of the sine wave.
Defaults to the height of the applet/2.
amplitudeintegerHeight of the sine wave in pixels.
Defaults to the height of the applet/4.
char heightintegerHeight of the text (1 = normal, 2 = double etc).
Defaults to 2.
char widthintegerWidth of the text (1 = normal, 2 = double etc).
Defaults to 1.
scroll speedintegerSpeed of the scroll (in pixels per frame).
Defaults to 4 pixels/frame.
scroll delayintegerMillisecond delay between frames.
Defaults to 100ms.
ripple speedintegerSpeed of the ripple effect (in pixels per second).
Defaults to 4 pixels/frame.
