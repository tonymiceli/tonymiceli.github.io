ArcTool

This was my largest program for the Amiga. It was written in straight 68000 assembly language and was basically a graphical frontend to archivers that hid the user from all those cryptic commandline switches. It featured a simple scripting mechanism so that support could be added for newer achivers without the need for a new release.

Since it was written for Workbench 1.3 to ensure as many Amigas could run it, most of the GUI components had to be written from scratch.
